🧭 How to Interpret

📌 p < 0.05 → Strong evidence against H₀ → Consider rejecting
📌 p ≥ 0.05 → Weak evidence → Keep H₀ (for now)
📌 If μ₀ is outside the CI → H₀ is unlikely
📌 Type I error = false alarm (reject true H₀)
📌 Type II error = missed effect (accept false H₀)
